# RDK - Roblox Development Kit

A utility library for Roblox development written in Luau.

## Installation

Add RDK to your `wally.toml`:

```toml
[dependencies]
RDK = "kolchanov-dev/rdk@0.1.0"
```

Then run:

```bash
wally install
```
